# learning-journal-wk5
# Learning Activities & Resources

This week I set up a local WordPress development environment for my Quiet Confidence website. I initially attempted to install Docker but experienced configuration issues, so I decided to use LocalWP, which provided an easier way to run WordPress locally with an integrated web server and database. I also created a WordPress child theme based on the Astra theme and customised the design using CSS. A child theme allows developers to modify the appearance of a site without editing the parent theme directly, ensuring changes are not lost when the theme is updated.
Resources used: LocalWP, WordPress Dashboard experimentation, class materials on local hosting.
http://quiet-confidence.local/

# Estimated Hours
I spent approximately 6–8 hours completing this task. This included attempting to install Docker, troubleshooting installation issues, setting up LocalWP, creating the child theme, implementing CSS customisations, and testing the site locally.

# Content Insights
This practical helped me better understand how WordPress themes and dynamic websites function. Creating a child theme demonstrated how developers can safely customise a website’s design while keeping the original theme intact. Using LocalWP also made it easier to focus on building and testing the site rather than resolving complex server setup problems.
